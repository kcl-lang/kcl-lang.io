## 单位的常量

- 定点数: `n`, `u`, `m`, `k`, `K`, `G`, `T` 和 `P`.
- 2 的幂: `Ki`, `Mi`, `Gi`, `Ti` 和 `Pi`.

## 函数列表

  将 int 转换为以 `n` 作为后缀的字符串
